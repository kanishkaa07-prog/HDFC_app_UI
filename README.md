# HDFC_app_UI
A modern UI/UX redesign of the HDFC Corporate Banking App developed during the Xebia Virtual Training Program. Designed in Figma, the project focuses on improving usability, accessibility, visual consistency, and streamlining corporate banking workflows through a clean, user-centered interface.
